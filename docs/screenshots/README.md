# Screenshots

Drop your own captures here so they show up in the main `README.md`.

Expected files (referenced by the README):

| File           | What to capture                                              |
| -------------- | ----------------------------------------------------------- |
| `swagger.png`  | The interactive Swagger UI at `http://127.0.0.1:8000/docs`  |
| `dashboard.png`| The web frontend dashboard after signing in                 |
| `appointments.png` | The appointments table (or the available-slots view)    |

## How to capture

1. Start the server: `uvicorn main:app --reload`
2. Open http://127.0.0.1:8000/docs (Swagger) or http://127.0.0.1:8000 (frontend)
3. Take a screenshot (macOS: `Cmd + Shift + 4`, then drag).
4. Save it here with the exact filename from the table above.
